# Proyek Analisis Data: Bike Sharing Dataset

## Deskripsi
Proyek ini bertujuan untuk menganalisis pola peminjaman sepeda berdasarkan musim, kondisi cuaca, dan waktu menggunakan Bike Sharing Dataset.

## Pertanyaan Bisnis
1. Bagaimana pengaruh musim dan kondisi cuaca terhadap jumlah peminjaman sepeda?
2. Bagaimana pola peminjaman sepeda berdasarkan waktu (jam) dan jenis hari?

## Insight
- Peminjaman sepeda tertinggi terjadi pada musim Fall dan kondisi cuaca cerah
- Terdapat pola jam sibuk pada pagi dan sore hari
- Hari kerja memiliki intensitas peminjaman lebih tinggi dibanding hari libur
- Pengguna registered lebih dominan dibanding casual
